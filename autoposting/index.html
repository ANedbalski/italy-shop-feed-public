AutoPosting MCP

Плагин для публикации товаров на italy-shop.eu через Claude Desktop.

v0.1.0 Python 3.11+ Claude Desktop

Быстрая установка (macOS/Linux)

curl -fsSL https://feed.italy-shop.eu/autoposting//install.sh | bash

Ручная установка

  1. Скачай autoposting_mcp-0.1.0-py3-none-any.whl
  2. Создай venv: python3 -m venv ~/.autoposting-mcp
  3. Установи: ~/.autoposting-mcp/bin/pip install autoposting_mcp-0.1.0-py3-none-any.whl
  4. Добавь в Claude Desktop config (см. ниже)

Claude Desktop Config

{
  "mcpServers": {
    "autoposting": {
      "command": "~/.autoposting-mcp/bin/python",
      "args": ["-m", "autoposting"],
      "env": {
        "SKYSHOP_API_KEY": "YOUR_API_KEY"
      }
    }
  }
}

Инструменты

  • process_images — сжатие фото в WebP
  • get_skyshop_options — цвета и размеры из Sky-shop
  • get_categories — дерево категорий
  • get_brand_catalog — каталог бренда
  • publish_product — публикация товара
